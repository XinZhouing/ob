# 部署环境准备
## mysql
- 使用 mysql 5.7 镜像
- 后台运行
- 数据持久化
- 发布端口 33060:3306
- 设置 root 密码
- 停止后自动重启
- 容器命名为 mysql5.7
### 命令
`docker run -d -v mysql5.7_data:/var/lib/mysql -p 33060:3306 -e MYSQL_ROOT_PASSWORD=root --restart unless-stopped --name mysql5.7 mysql:5.7`
## redis
- 使用 redis 7镜像
- 后台运行
- 数据持久化
- 发布端口 6379:6379
- 停止后自动重启
- 容器命名为 redis 7
- 开启 redis 的时候要开启 aof 持久化
